# BaePo

Best Pommes in Town!

  

# To run:

Do a **`docker compose up`** in the project folder *baepo3/my-baepi*

  

Go to the next.js directory *baepo3/my-baepi/nextjs* and run these commands:

- **`npm install`**

- **`npm run build`**

- **`npm run start`**

 Go to: http://localhost:3000
 
**Enjoy your Pommes!**
